# 键盘和鼠标快捷键

以下是 Arrow 编辑器的键盘和鼠标快捷键列表：

## 检查器面板

要在*检查器*面板中打开节点，只需`双击`任意节点，
或者如果*自动检查*快速偏好设置已激活，则只需选择一个即可。

+ `Ctrl + I` 切换*自动检查*。

你还可以使用状态栏的*快速偏好设置*菜单按钮来管理类似行为。

与检查器面板相关的其他有用快捷键如下：

+ `Ctrl + Shift + Minus (-)` 将参数重置为检查节点的最后保存状态
+ `Ctrl + Shift + Plus (+)` 更新节点参数
+ `Ctrl + P` 切换检查器面板
+ `Ctrl + \` 更改网格视图以聚焦到检查的节点

+ `Ctrl + U` 切换*自动节点更新*

*自动节点更新*快速偏好设置（默认启用）在*用户离开节点时*自动应用节点修改。
即通过检查另一个节点或通过点击网格或多选取消选择当前节点。

> 此行为仅适用于节点。其他资源必须手动设置以更新。
> 要永久存储数据，应保存文档。

如果启用了*重新检查时重置*快速偏好设置，重新选择节点将*重置*修改的参数为最后保存的状态。

### 项目列表

提供移动或排序功能的项目列表（例如*对话*和*交互*列表），
也支持以下快捷键：

+ `Home` 将选中项移动到顶部
+ `End` 将选中项移动到末尾
+ `PageUp` 或 `Ctrl + Up` 将选中项向上移动
+ `PageDown` 或 `Ctrl + Down` 将选中项向下移动
+ `Delete` 如果允许则删除选中项

## 网格（图形）和编辑器

+ `Ctrl + Arrow-Keys` 在（聚焦的）网格上移动选中的节点

+ 选择
  + `Ctrl + W` 取消选择所有节点
  + `Ctrl + Left-Click` 用于选择多个节点
  + `Shift + Left-Click` 选择两个选中节点之间的分支（即一系列节点）
      > 它接受一个起始节点和多个结束节点。
      > 你可以按住 `Alt` 激活瀑布模式，以同时选择系列的兄弟节点。

+ 查询（全局文本搜索）
  + `Ctrl + Q` 或 `F3` 聚焦到查询（搜索）输入框
  + `Ctrl + 2` 在网格上定位下一个匹配项
  + `Ctrl + 1` 定位上一个匹配项

+ 视图
  + `F11` 切换全屏
  + `鼠标滚轮` 放大或缩小，`Ctrl + Plus|Minus` 也可以
  + `鼠标中键` 抓取网格并移动视图
  + `Ctrl + 鼠标滚轮` 垂直滚动网格，如果结合 `Shift` 则水平滚动
  + `Ctrl + 0` 重置缩放

## 控制台面板

以下快捷键用于启动*控制台*面板并从特定节点向前播放：

+ `F4` 清除控制台后从*项目*的活动入口播放
+ `F5` 清除控制台后从*场景*的活动入口播放
+ `F6` 以关闭时的状态打开控制台
+ `F7` 不清除控制台的情况下播放最后选中的节点（如果有）

## 项目管理

+ `Ctrl + S` 保存打开的文档
+ `Ctrl + E` 使用最近导出的路径和格式快速重新导出（如果有）

## 修订/历史管理

+ `Ctrl + D` 拍摄快照
+ `Ctrl + Z` 撤销最后应用的更新
+ `Ctrl + Shift + Z` 重做最后应用的更新

> 历史行为作为一个尚未完善的功能，默认不激活。
> 要启用它，请从*偏好设置*面板增加历史大小。
