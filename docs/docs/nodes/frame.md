# 框架节点 (Frame)

*Frame*（框架）节点被设计为 Arrow 编辑器的网格辅助工具。

它们是可调整大小的矩形，用于对覆盖在其表面区域下的其他网格节点进行分组。

Frame 有一个 `label`（标签）来提供关于其目的或内容的信息，以及一个 `rect`（即 [宽度, 高度]）和一个 `color`（颜色），定义它们的视觉外观和功能。

### 约定

+ Frame 节点不接受入站或出站连接，但可以是 [Jump](./jump) 的目标。在这种情况下，它们的行为类似于行尾 [Marker](./marker) 节点。这不是 Frame 的推荐或预期行为，而是一种后备，因此如果观察到跳转到 Frame，运行时实现可能会抛出警告。

### 使用

Frame 像其他节点一样被添加到网格中。

它们可以从右下角调整大小。

Frame 节点在鼠标离开时尝试移动并保持在其包裹的节点后面。它们还可以使用右上角的按钮临时折叠。

Frame 节点独立移动。要同时移动其所有内容，请在其顶栏或没有其他节点覆盖的表面上使用 `Ctrl + Alt + 双击`。

在被覆盖的节点上 `Alt + 双击` 将选择该节点。结合 `Ctrl` 可以选择多个被覆盖的节点。
