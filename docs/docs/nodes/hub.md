# 集线器节点 (Hub)

*Hub*（集线器）节点是非常简单但有用的导航节点，通过接受多个入站连接（网格图链接）并仅播放一个出站槽来合并多个分支。

它们唯一的参数是其入站 `slots`（槽）的数量。

### 约定

+ Hub 是自动节点：它不等待用户交互。

+ 考虑到其自然目的，Hub 节点预期至少有两个入站槽。

+ Hub 即使被跳过也会履行其存在的职责，并播放其唯一的出站槽。

### 使用

合并分支或出站链接是 Hub 节点存在的唯一简单任务。

> 如果您打算将分支合并到另一个场景或网格上远处的节点，[Jump](./jump) 节点被证明非常干净和有用。
